1. Bootstrap serve per creare siti senza dover fare il CSS.
2. <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/css/bootstrap.min.css" rel="stylesheet">>
3. Un CDN (Content Delivery Network) è utilizzato per caricare file (Comunemente file CSS) senza doverli scaricare e quindi perdere tempo.
4. content="width=device-width, initial-scale=1 Serve per controllare il comportamento della visualizzazione del contenuto in base alla grandezza del dispositivo.
5. La classe .container ci da' un contenitore con la larghezza fixed e responsive, il .container-fluid ci da' un contenitore con la larghezza the occupa tutta l'asse X del viewport, non è responsive
